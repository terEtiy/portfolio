# Portfolio — terEtiy

Набор проектов: веб-инструменты, плагины для QGIS, Python-скрипты и расширения Chrome.

---

## Проекты

### GPZU Assistant v2
 - Node Editor v3	Визуальный редактор потоков данных: создание узлов, соединение линиями, передача XML через цепочку узлов, сохранение сценариев
 - AST Visualizer	Парсит JavaScript код и строит интерактивный граф синтаксического дерева (AST)
 - Image Comparator	Редактор слоёв изображений — загрузка, настройка прозрачности
 - Редактор БД	Просмотр и редактирование базы данных ограничений (работает в связке с "Генератором готовых ограничений")
 - Генератор готовых ограничений -	Генерирует готовые тексты ограничений, для ГПЗУ, по шаблонам, копирует в буфер
 - Генератор 2.3	Специализированный генератор для конкретного раздела документации ГПЗУ
 - XML парсер	Парсит XML, строит таблицу: тег + содержимое + глубина вложенности + атрибуты
 - PDF → JPG	Конвертирует каждую страницу PDF в JPG, упаковывает в ZIP
 - Координаты из XML ЗУ	Извлекает координаты земельных участков из XML, отображает на интерактивной карте	Leaflet.js

### tool
QGIS плагин для работы с геопространственными данными. Инструменты загрузки и выгрузки слоёв, маппинг полей.

### Мониторинг задач v2
Веб-приложение для мониторинга и управления задачами. Python-бэкенд + HTML/CSS/JS фронтенд.

### Python-скрипты для QGIS
- `Parser_restriction.py` — парсер ограничений
- `Poligon_intersects V3 for qgis 3-6-3.py` — поиск пересечений полигонов
- `XML_parser_OKS and NUM.py` — парсер XML для ОКС
- `cordinate_print_Q3.py` — вывод координат
